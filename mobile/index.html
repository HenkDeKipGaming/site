Een plugin gemaakt voor Minetopia, een Minecraft gamemode van DusDavidCraft.

MinetopiaATM zorgt voor een geheel banksysteem inclusief Banen, rangen, scoreboards en nog veel meer. Deze plugin werkt samen met Vault en Essentials. Deze plugins heb je dus nodig om MinetopiaATM te laten werken. Van deze plugin is een GRATIS versie met om de 10 minuten een reclamebericht en een PREMIUM versie waar die reclame niet in zit. PvPWhitelist

Whitelist items waarme spelers kunnen slaan!

Deze plugin is speciaal gemaakt voor Minetopia servers. In Minetopia servers hebben owners vaak het probleem dat ze de politie permissie moeten geven om te slaan. Met deze plugin kan je gemakkelijk items aan de whitelist toevoegen en er uit halen. Je kan ook een speler toevoegen aan de whitelist zodat deze speler met alle items kan vechten.
